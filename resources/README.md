# Visual resources

Put curated interactive screenshots or short GIFs from the renderer in this directory. Prefer descriptive filenames so the README can pick representative images quickly, for example:

- `classic_boids_overlay.png`
- `birdflight_flock.png`
- `fishschool_depth_view.gif`
- `noise_experiment_overlay.png`

Keep large raw captures out of Git; commit only presentation-ready images that should appear in the portfolio README.
